# Nippard BTS — 3-Day Upper Focus

A personal workout tracker based on Jeff Nippard's **Bodybuilding Transformation System (Beginner)**, adapted to a 3-day/week schedule with an upper body strength emphasis.

🔗 **[Open the app](https://jaanus91.github.io/nippard-plan)**

---

## Schedule

- **Start:** 07 May 2026 · **End:** 23 Aug 2026
- **Days:** Tuesday / Thursday / Sunday (week 1 special: Thu / Sat / Sun)
- **Rotation:** Upper (strength) → Pull (strength) → Push (hypertrophy) → Legs (hypertrophy)
- **16 calendar weeks · 48 total sessions**

### Training Blocks

| Cal weeks | Block | RPE |
|---|---|---|
| 1–2 | Foundation | 6–7 |
| 3 | Transition | 6–8 |

| 7 | Transition | 7–9 |
| 8–12 | Intensification | 8–9 |
| 13 | Deload | 5–6 |
| 14 | Transition | 6–9 |
| 15–16 | Peaking | 9+ |

## Features

- **Auto week detection** — opens on the current calendar week automatically
- **Next session banner** — shows your next workout date and countdown
- **Day cards** — click any day to expand the full exercise table with sets, reps, RPE and rest times
- **RPE tooltips** — hover any RPE value for a plain-language explanation
- **YouTube links** — every exercise links directly to a demo video
- **Substitutions** — two alternatives listed for every exercise
- **Warm-up protocol** — collapsible general + exercise-specific warm-up guide
- **16-week timeline** — colour-coded block overview, click any week to jump
- **Arrow key navigation** — ← → between weeks
- **Print button** — clean print layout for any week

---

## Modification from original program

The original BTS Beginner program runs 5 days/week. This version removes the **Lower (Strength)** workout, leaving a 4-workout rotation run over 3 days/week, extending the programme from 12 to 16 calendar weeks.
